CALLBACK SCRIPT TESTING PROCEDURE:-
---------------------------------
Please follow the below mentioned steps to call the callback script for all the payment types.

Url: <siteurl>/<callaback script file name>
	[Ex: https://woocommerce.novalnet.de/callback_novalnet2woocommerce.php]

Note:
-----

Set Email Configuration

	//Reporting Email Address Settings

For testing callback script, kindly make the below following changes in the file "callback_novalnet2woocommerce.php"
----------------------------------------------------------------------------------------------------------------------------
 
 1) $debug = true; 

 2) $test  = true; 

 3) a) formatted url for Invoice / Prepayment :-
	
	<Site URL>/callback_novalnet2woocommerce.php?vendor_id=4&status=100&payment_type=INVOICE_CREDIT&tid_payment=12675800001204435&amount=3778&tid=12675800001204435&order_no=12

		Payment type:
		-------------

		INVOICE_CREDIT (Invoice and Prepayment)

    b) formatted url for other payment types which are mentioned below: (except Invoice / Prepayment)

	<Site URL>/callback_novalnet2woocommerce.php?vendor_id=4&status=100&payment_type=PAYPAL&amount=3778&tid=12675800001204435&order_no=12

		Payment types: 
		--------------

		PAYPAL (PayPal)
		ONLINE_TRANSFER (Instant Bank Transfer)
		CREDITCARD (Credit Card)
		IDEAL (iDEAL)
		DIRECT_DEBIT_SEPA (Direct Debit SEPA)

Please use the respective payment type in the param "payment_type". (EX : payment_type=PAYPAL) 


   Parameters:
   -----------
	tid		Callscript Transaction TID
	vendor_id	Merchant ID
	status		Successfull payment transaction value
	payment_type	Types of payment process
	tid_payment	Existing appropriate order Transaction ID (only for Invoice/Prepayment payment methods)
	amount		Customer paid amount in cents
	order_no	Existing appropriate order number

For real time transactions, kindly do the following changes in the file "callback_novalnet2woocommerce.php"
-------------------------------------------------------------------------------------------------------------------

 1) $debug = false; 
 
 2) $test  = false;
